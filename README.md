# open-admin-process

## 简介
开源后台流程管理项目。

## Maven 引入方式

在你的 `pom.xml` 中添加如下依赖：

```xml
<dependency>
    <groupId>io.github.jiangood</groupId>
    <artifactId>open-admin-process</artifactId>
    <version>版本号</version>
</dependency>
```

该库已发布于 [Maven Central](https://search.maven.org/artifact/io.github.jiangood/open-admin-process)。
